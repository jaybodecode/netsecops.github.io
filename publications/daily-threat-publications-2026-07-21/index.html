Ransomware Surges, Zero-Days Emerge, AI Fuels Attacks

Ransomware Surges, Zero-Days Emerge, AI Fuels Attacks

July 21, 2026
10 articles (4 new, 6 updated)
30 min read

Summary

This daily cybersecurity summary highlights a significant increase in ransomware attacks, particularly targeting European supply chains, with a 55% surge reported. The ransomware ecosystem is fragmenting, with new groups emerging rapidly. Government agencies are also experiencing daily ransomware incidents, though some have successfully resisted demands. A critical Windows zero-day, 'LegacyHive,' has been patched by 0Patch, granting local attackers SYSTEM privileges. SonicWall SMA devices are vulnerable to zero-day exploits leveraged by the Inc Ransomware group, allowing for command execution and malware deployment. The AI-powered ransomware 'JadePuffer' is now deploying 'ENCFORGE' to target and destroy machine learning models, exploiting critical vulnerabilities in Langflow. The healthcare sector remains vulnerable, with Craneware experiencing a data breach, underscoring supply chain risks. In the UK, 26% of businesses have faced supply chain attacks, with many continuing to work with insecure suppliers. On the defensive front, Cisco has released open-weight AI models, 'Antares,' for code analysis to aid in vulnerability detection. However, attackers are also leveraging AI, with an exposed server revealing an AI-powered phishing toolkit targeting Windows users in Mexico. Furthermore, China-aligned APT groups are integrating public AI tools into espionage campaigns, compromising systems in Thailand, Afghanistan, and Taiwan. Defenders should review behavior-based detection, attack surface management, and credential rotation strategies in light of these evolving threats.

Supply Chain Attacks Hit 26% of UK Businesses in the Last Year

MEDIUM

1 in 4 UK Firms Hit by Supply Chain Attacks

A survey by Databarracks reveals that one in four (26%) UK businesses have suffered a cyber incident originating from their supply chain in the past 12 months. The 'Data Health Check 2026' report highlights a concerning trend: 48% of IT decision-makers admit to knowingly working with suppliers that have security or resilience issues. This is often driven by a strong dependence on these suppliers, which 26% of respondents cited as a major barrier to improving their own organization's security posture. The findings underscore that while awareness of supply chain risk is high, practical and business constraints often prevent organizations from effectively mitigating it, even as cyberattacks remain the top cause of IT downtime.

Cisco Releases Open-Weight 'Antares' AI Models for Code Analysis

INFORMATIONAL

Cisco Releases Antares AI for Code Vulnerability Detection

Cisco has released Antares, a family of open-weight small language models (SLMs) specifically trained to perform static application security testing (SAST) and identify security vulnerabilities in source code. The models, including Antares-1.3B and Antares-7B, are designed to run locally on standard hardware, making AI-powered security analysis more accessible to developers and security practitioners. Cisco emphasizes that Antares is intended to augment existing security workflows by accelerating the initial triage of potential flaws, not to replace comprehensive security tools or human review. The models are significantly faster than larger counterparts like GPT-4, aiming to democratize the use of AI in daily security operations.

Exposed Server Reveals Attacker's AI-Powered Phishing Toolkit

MEDIUM

AI-Assisted Phishing Toolkit Found on Exposed Server

Researchers at Rapid7 discovered an unsecured server that exposed a sophisticated, AI-assisted toolkit for phishing and malware delivery. The operator, believed to be Russian-speaking, was targeting Windows users in Mexico with an infostealer. The toolkit shows extensive use of Large Language Models (LLMs) to generate phishing lures, create exploit test plans, and manage campaigns, a workflow Rapid7 describes as 'LLM-assisted.' The server's admin panel, left with default credentials, logged over 77,000 requests in one week, primarily from Mexico. The findings provide a rare look into how cybercriminals are adopting AI to increase the efficiency and sophistication of their operations, treating them like modern software projects.

China-Aligned APTs Use Public AI Tools in Espionage Campaigns

HIGH

Chinese APTs Leverage Public AI Tools in Attacks

State-sponsored threat actors aligned with China are integrating publicly available AI tools, such as Anthropic's Claude Code and DeepSeek, into their cyber-espionage campaigns. A report from Hunt.io details how the APT groups used these tools as core components to automate attacks, generate malicious scripts, and execute commands. The campaigns successfully compromised government systems in Thailand and Afghanistan, as well as financial organizations in Taiwan. This tactic of 'living off the trusted service' allows attackers to blend in, as the AI tools may already be approved for use within target networks, marking a significant evolution in APT tradecraft.

New 'LegacyHive' Windows Zero-Day Exploit Grants Admin Access

HIGH

LegacyHive Zero-Day Exploit for Windows Privilege Escalation

Update:

Security firm 0Patch has released free, unofficial micropatches to mitigate the 'LegacyHive' Windows zero-day vulnerability. This provides immediate protection while Microsoft investigates. The exploit's mechanism has been further clarified: it allows a local attacker to gain NT AUTHORITY\SYSTEM privileges by manipulating another user's NTUSER.DAT registry hive. This enables arbitrary code execution upon the next administrator login, a significant escalation from previously understood data read access. All modern Windows versions remain affected, and the PoC is public.

Inc Ransomware Exploits Critical SonicWall SMA Zero-Day Vulnerabilities

CRITICAL

Inc Ransomware Exploits SonicWall SMA Zero-Days

Update:

Further analysis reveals attackers leveraged the SSRF (CVE-2026-15409) to access a local CouchDB service configured with default 'admin:admin' credentials. This access was then used to inject and execute arbitrary commands (CVE-2026-15410) with root privileges. The deployed KNUCKLEBALL malware is noted to be memory-resident, injecting malicious components directly into legitimate SonicWall processes to evade file-based detection and maintain stealthy, long-term persistence for credential theft and network monitoring.

AI-Powered Ransomware 'JadePuffer' Deploys 'ENCFORGE' to Destroy ML Models

HIGH

JadePuffer AI Ransomware Targets ML Models with ENCFORGE

Update:

Further analysis of the ENCFORGE campaign reveals that CVE-2025-3248, the initial access vulnerability in Langflow, has a critical CVSS score of 9.8 and affects versions prior to 1.3.0. This update also notes that CISA has previously added other Langflow RCEs to its Known Exploited Vulnerabilities (KEV) catalog, underscoring the framework's status as a high-priority target for threat actors. Additional MITRE ATT&CK techniques identified include T1059.006 (Python execution) and T1562.001 (Disable or Modify Tools). A critical mitigation step added is the immediate rotation of all secrets, API keys, and credentials accessible to the Langflow process, as they should be considered compromised post-exploitation.
